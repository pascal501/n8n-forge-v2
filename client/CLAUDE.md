# Frontend N8N Forge V2 (client/)

Ce dossier contient le frontend de N8N Forge V2 (React). Quand tu travailles ici, concentre-toi uniquement sur le code frontend.

## Structure importante

- `src/App.jsx` : point d'entrée principal de l'application.
- `src/components/` : composants réutilisables.
- `src/pages/` (si présent) : pages principales.
- `src/hooks/` (si présent) : hooks personnalisés.

Si tu as besoin de plus de détails, commence par lire `src/App.jsx` puis les fichiers que je te mentionne dans mes demandes.

## Comment je veux que tu travailles dans client/

- Proposer des changements **par petites étapes**, sur des fichiers précis.
- Quand tu modifies du code :
  - indiquer clairement le nom du fichier,
  - montrer uniquement les morceaux de code à changer,
  - expliquer rapidement ce que tu fais et pourquoi.
- Ne pas réécrire des fichiers entiers si ce n'est pas nécessaire.
- Garder le style existant (formatage, nommage, façon d'organiser les composants).
- Préférer des composants simples et lisibles plutôt que des grosses fonctions complexes.

## Ce qu'il faut éviter

- Éviter de toucher à trop de fichiers en même temps dans une seule réponse.
- Ne pas introduire de dépendances lourdes sans me demander avant.
- Ne pas inventer de structure de dossiers : respecter ce qui existe déjà.
