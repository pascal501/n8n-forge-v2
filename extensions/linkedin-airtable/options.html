LinkedIn → Airtable

Configurez vos identifiants Airtable pour enregistrer des contacts LinkedIn.

Authentification Airtable

Créez votre token sur airtable.com/create/tokens avec les scopes data.records:write et schema.bases:read.

Base & Table

Visible dans l'URL de votre base Airtable : airtable.com/appXXX/…

Dans l'URL : airtable.com/appXXX/tblXXX/…

Résumé IA (optionnel)

Génère un résumé du profil dans « Profile Summary ». Créez une clé gratuite sur aistudio.google.com/apikey. Laissez vide pour désactiver.

Robot d'enrichissement (mode batch)

Temps de pause entre deux profils LinkedIn pour imiter un rythme humain et éviter le bannissement. Recommandé : 45s (minimum 10s).

Le robot traite au maximum 200 profils consécutifs (sécurité LinkedIn). Pour un premier test, mettez 5.
